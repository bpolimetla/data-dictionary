
How to run DDGen.jar file?

Step 1:
Download and copy following files to ddgen folder
log4j-1.2.12.jar
ojdbc6-11.1.0.7.0.jar

http://mvnrepository.com/artifact/log4j/log4j/1.2.12

http://www.mvnsearch.org/maven2/com/oracle/ojdbc6/11.1.0.7.0/
https://maven.nuxeo.org/nexus/content/groups/public/com/oracle/ojdbc6/11.1.0.7.0/


If exact version is not available, copy similar version files and modify run.bat file.

Step 2:
Modify java home and other versions in run.bat file.

Step 3: Enter correct jdbc connection string 
DDGen.properties

Step 4: In Windows
Run run.bat file from command prompt to see the result files.

For Unix/Linux perform the similar steps.
